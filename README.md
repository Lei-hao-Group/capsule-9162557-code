## Overview

This repository contains the source code and analytical framework for the paper: **"[Direct and Indirect Effects of Influenza Vaccination on the Risk of Infection with Multiple Respiratory Pathogens in Children]"**.

The code implements a data analysis pipeline including data preparation, a matched case-control study, and a disease transmission model simulation.
